* Arrays :
  -----
   int[]
    -> fixed size allocated when defined. 
   int[][]
    -> 2D array  (We need to define size at root level & can make it dynamically sized)

   Array.toString()

* ArrayList :
  --------
   -> arrays i.e [] are of fixed size 

   -> ArrayList are of Dynamic size   (It's an impl of List interface using Array DS)
      (equi to Vector in c++)

   -> Cannot index on ArrayList, use `get()`, instead
